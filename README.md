# Zijun Wang

**I build production AI systems and study where their value lands.**

Technical background in NLP and machine learning. I ship LLM applications end to end, and I analyze AI as both a technological system and an economic shift — what models can actually do, and where the value they create ends up across the US and China ecosystems. Running these systems in production is where my research priors come from.

**Selected work**

- **[Inner Order OS](https://github.com/zijunVV/inner-order-os)** — production AI platform (Next.js · Supabase · Groq LLM). Three-layer output guardrails, evidence-based model selection, and privacy guards, built and deployed solo. [Live](https://inner-order-os.vercel.app)
- **[Stock Analyzer](https://stockanalyzer-ai.vercel.app/)** — live AI analysis platform covering US · HK · China A-share markets, with a curated [AI Value-Chain map](https://stockanalyzer-ai.vercel.app/value-chain) where every ticker carries a one-line thesis. FastAPI · React · Groq
- **[Bridge the Gap](https://github.com/zijunVV/bridge-the-gap)** — full-stack AI platform for cross-cultural communication, with a from-scratch content-authority ranking algorithm. [Live](https://bridge-the-gap-ai.vercel.app)
- **[KCAS · Knowledge Capital Ecosystem](https://github.com/zijunVV/kcas-core)** — four open-sourced frameworks spanning the capital lifecycle: evaluate → research → allocate → create

**Current research — *Who Captures Value in the AI Era?*** A layer-by-layer framework on the AI value chain (semiconductors → cloud → foundation models → agents → applications), built around two theses: *Scarcity → Control → Ownership* and *Value Creation ≠ Value Capture ≠ Attractive Investment*. Operationalized as a tracker with earnings predictions committed before each call and graded after. Research sample and deck available on request.
